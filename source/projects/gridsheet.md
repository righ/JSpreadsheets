#### Introduction
GridSheet is a high-performance, React-first spreadsheet UI library designed for modern web applications.

Unlike traditional data grid components, GridSheet focuses on spreadsheet-native capabilities such as built-in formula evaluation, automatic recalculation, and multi-sheet references. It supports cross-sheet formulas, enabling structured separation of input, calculation, and presentation layers.

GridSheet is built with extensibility at its core. Developers can customize renderers, parsers, styles, and functions — including asynchronous functions — making it suitable for dynamic, API-driven applications. Policy-based input control and clipboard customization allow fine-grained enforcement of business rules and security requirements.

Performance is a core design goal. GridSheet includes a public demo handling 10,000 rows × 100 columns (1,000,000 cells), demonstrating efficient rendering and large dataset handling.

Released under the Apache 2.0 License, GridSheet is free for commercial use and designed to serve as a spreadsheet engine UI for modern SaaS and application platforms.
